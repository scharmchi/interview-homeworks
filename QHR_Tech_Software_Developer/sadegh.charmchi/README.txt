Minimum prerequisites:
- An IDE that can work with Maven (e.g. Eclipse, NetBeans, etc.)

IDE setup:
- Open project at root folder.  pom.xml includes all dependencies you need as a base.

Development Instructions:
- Make your changes to the MySubmission class.
- Execute the Main Class of your code in the IDE
	- This will run an in-built test and print out your results as well as a transcript.
- When you are satisfied with your results, commit the transcript along with your code back into the repository provided.
- Contact QHR to let them know you are done.
